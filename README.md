# ☕ Interactive Coffee Sales Dashboard

## Overview
Cleaned and transformed raw sales data by resolving blank fields using VLOOKUP, then built a dynamic dashboard using pivot tables and charts.

Skills: Excel, VLOOKUP, Pivot Tables, Pivot Charts, Slicers, Timeline

## Features
- Total Sales Over Time - line chart tracking revenue across 4 coffee types
- Sales By Country - bar chart comparing performance across regions
- Top 5 Customers - bar chart highlighting highest-value customers
- Interactive Timeline - scrub through time to update all visuals dynamically
- Slicers - filter all charts simultaneously by roast type, bag size, and loyalty card status
